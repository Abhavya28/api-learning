# API Learning

This repository contains my hands-on practice and learning around APIs, REST concepts, and frontend API consumption.

The goal of this repository is to build a strong understanding of how frontend applications interact with backend services using modern JavaScript tools and patterns.

---

## What this repository covers

- REST API fundamentals
- HTTP methods (GET, POST, PUT, DELETE)
- API request and response flow
- Fetch API in JavaScript
- async / await for handling asynchronous operations
- Basic backend setup for learning purposes
- Frontend API consumption patterns

---

## Projects & Folders

### 1. rest-api
- Basic REST API setup using Node.js and Express
- Understanding API routes and JSON responses
- Practicing request handling and response status codes

### 2. fetch-axios-dashboard
- Frontend project built with React
- API consumption using Fetch API
- API consumption using Axios
- Managing loading and error states
- Clean separation of API logic and UI components
